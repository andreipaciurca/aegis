macOS · Linux · Windows — one static binary

Internet security, in your terminal.

aegis is a malware & ransomware scanner, firewall panel, network monitor and persistence auditor in a single fast TUI, CLI and local browser GUI. Use aegis app when you want the TUI and GUI paired together. AEGIS means Adaptive Endpoint Guard for Internet Safety. No daemons, no kernel hooks — it drives your OS's own security machinery and gets out of the way.

what it does

Seven panes, no magic.

scanner

SHA-256 hashing against a local signature database, an updatable YARA-lite rule engine (reverse shells, Mimikatz, PowerShell cradles, webshells, coinminers), and entropy analysis for packed binaries. One key quarantines a threat — chmod 000, JSON audit log — and aegis restore undoes it if you were wrong; nothing is ever silently deleted.

✗ CRITICAL loader.ps1 — PowerShell download-and-execute cradle
ransom shield

Deploys harmless canary files and watches them for tampering — if ransomware encrypts one, aegis knows in seconds. A sweep also catches ransom notes, known extensions (.lockbit, .wncry), and magic-byte mismatch: a “.jpg” that isn't a JPEG and reads as encrypted.

✗ CRITICAL canary encrypted (entropy 8.0) — active ransomware likely
persistence audit

Enumerates the autostart mechanisms malware abuses to survive reboot — LaunchAgents, systemd, cron, autostart, Run keys — and flags suspicious entries (temp-dir payloads, encoded commands), each with the exact command to disable it.

⚠ LaunchAgent com.x.helper — runs from a temporary directory
network + remediation

A live table of every connection and listener, suspicious entries first. Then act: select a row and press k to kill the process (with a confirm prompt) or b to block the port at the native firewall — the exact command shown either way.

⚠ listening on all interfaces (:5000) → press k / b
firewall

Reads and toggles the native OS firewall instead of reinventing one: macOS Application Firewall + pf + stealth mode, Linux ufw / nftables / iptables, Windows Defender Firewall. Without privileges it prints the exact sudo command instead of failing.

● ENABLED · stealth on · 42 rules
updates

Startup refreshes MalwareBazaar and URLhaus payload hashes, checks whether a newer aegis release exists, and checks the latest llama.cpp release. Press u in the TUI/GUI or run aegis update from a script — same maintenance bundle either way; binary and model updates are reported, never silently replaced.

✓ signatures +633 (634 total) · Aegis current · llama.cpp b10068
osint

When you explicitly ask, aegis can enrich a hash with VirusTotal API v3 file reputation. Normal scans never upload files or call external reputation services; intel sends only the MD5, SHA-1 or SHA-256 you provide.

aegis intel <sha256> --json
clamav bridge

For a free self-hosted AV engine, run clamd locally and let aegis stream files with ClamAV's INSTREAM protocol. No vendor upload, no bundled ClamAV code or database, just explicit local scanning.

aegis clamav ~/Downloads --json

how it works

One binary, layered defence.

aegis adds no machinery of its own. Each subsystem is a thin, fast layer over a primitive your OS already ships — so the footprint stays tiny and there's nothing left running when you quit.

Inside a scan — cheapest checks first

Every file runs a short-circuiting pipeline: the fast, free checks reject or clear most files before aegis ever reads the whole thing to hash it.

1Filename heuristics — ransomware extensions, ransom notes, double-extension masquerades, temp/hidden executablesname only
2Signature hash — full-file SHA-256 matched against the local abuse.ch database1 read
3Rule engine + entropy — YARA-lite string/hex rules and Shannon entropy over a 64 KB head64 KB
4Magic-byte mismatch — extension promises a format the bytes don't deliver, at high entropy = likely encrypted16 bytes
✓ cleankeep walking — nothing written, nothing touched
✗ threatreported with severity · one key to quarantine (chmod 000, audit-logged)

quickstart

Running in under a minute.

# macOS / Linux install or updatecurl -fsSL https://raw.githubusercontent.com/andreipaciurca/aegis/main/scripts/install.sh | sh
# Windows PowerShell install or updateiwr https://raw.githubusercontent.com/andreipaciurca/aegis/main/scripts/install.ps1 -UseB | iex
# or build from source (Go 1.21+)git clone https://github.com/andreipaciurca/aegis && cd aegis && make install
# launch the TUI, GUI, or both paired togetheraegis aegis gui aegis app
# or script it — exit 1 means threats foundaegis update && aegis scan ~/Downloads && aegis shield && aegis audit
# optional: free local ClamAV daemon checkaegis clamav ~/Downloads --addr tcp://127.0.0.1:3310
# optional: local llama.cpp analyst for explanationsaegis ai config --backend llamacpp-server --endpoint http://127.0.0.1:8080/v1/chat/completions aegis scan ~/Downloads --ai
# verify detection safely with the EICAR test fileprintf 'X5O!P%@AP[4\PZX54(P^)7CC)7}$EICAR-STANDARD-ANTIVIRUS-TEST-FILE!$H+H*' > /tmp/eicar.txt aegis scan /tmp

local analyst

Optional AI explanations, local by default.

aegis can talk to a local GGUF model through llama.cpp, and can also use an explicit OpenAI-compatible API backend when the user configures a key. Local remains the default privacy path; remote model access is opt-in.

guided setup

aegis ai setup queries the latest llama.cpp release at runtime, recommends a small Gemma GGUF, and can download/extract the matching macOS, Linux or Windows asset with checksum verification.

aegis ai setup --download-llama
false-positive triage

Detections stay deterministic. The model explains why a file, process or persistence entry was flagged, estimates false-positive likelihood and suggests safe next steps.

aegis scan ~/Downloads --ai
privacy boundary

By default aegis sends metadata only: path, size, hash, severity and reason. Tiny printable excerpts are opt-in for better text/script triage.

aegis ai config --privacy excerpt --max-excerpt 2048
remote opt-in

Users can connect ChatGPT-style OpenAI-compatible APIs by choosing a backend, endpoint, model and API-key environment variable. Cloud use is visible and never the default.

aegis ai config --backend openai-compatible --remote-model gpt-5-mini --api-key-env OPENAI_API_KEY
useful companion

A chat or small status companion can summarize checkups, explain new KEV/NVD items and use approved local notes. It should learn as local context, not by silently retraining on private files — and never by replacing the deterministic detections above.

aegis ai remember "Port 5000 is expected on this dev box" aegis ai chat

VirusTotal and ClamAV are covered above under osint and clamav bridge — the AI analyst never triggers either on its own. Roadmap notes on durable storage and team threat-intel connectors live in the README.

Honestly: aegis is a capable lightweight tool, not a drop-in replacement for Bitdefender or Kaspersky — those run kernel drivers, cloud ML and paid research teams. Keep Gatekeeper, XProtect or Defender on. Signatures catch known samples; rules, entropy and heuristics catch common patterns; canaries catch ransomware as it acts. No kernel hooks, no traffic interception, no on-access scanning — by design, to stay fast. Optional VirusTotal lookups send only the hash you type and never run during normal scans.

Trust: security tools can trigger antivirus heuristics. Releases should ship checksums, signatures and platform code signing where available. See the trust & verification page for verification steps and false-positive submission guidance.
